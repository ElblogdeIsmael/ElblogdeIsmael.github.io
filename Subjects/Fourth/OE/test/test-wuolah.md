1. Considerando la organización como un sistema de flujos regulados, señale la afirmación correcta:

    - (x) La visión de la organización como un sistema de flujos regulados considera, entre otros, el movimiento de material e información que tiene lugar dentro del núcleo de operaciones, denominado como flujo de trabajo de operaciones.
    - ( ) Los flujos regulados se relacionan con el mecanismo de coordinación de la adaptación mutua.
    - ( ) La visión de la organización como un sistema de flujos regulados considera, entre otros, a los círculos prácticamente independientes de individuos que comparten responsabilidades sobre un conjunto de decisiones adecuadas a su particular nivel jerárquico, denominados como constelaciones de trabajo.
    - ( ) La visión de la organización como un sistema de flujos regulados solo considera a las relaciones de autoridad que tienen lugar dentro de la organización.

    > Correcta: el flujo de trabajo de operaciones. Los flujos regulados (operaciones, información-decisión de control y staff) se asocian a la normalización, no a la adaptación mutua; las constelaciones de trabajo pertenecen a la visión de comunicación informal.

2. Un puesto de trabajo que está especializado horizontalmente:

    - ( ) Siempre estará muy formalizado.
    - ( ) Siempre estará ampliado verticalmente.
    - ( ) No puede exigir formación para ser realizado.
    - (x) Realiza pocas tareas o tareas con poco contenido.

    > Correcta: realiza pocas tareas. La especialización horizontal se refiere al número de tareas (pocas y de escaso contenido), independiente del control vertical o de la formalización.

3. Una organización en la que todo el poder recae en los profesionales del núcleo de operaciones se encuentra:

    - ( ) Descentralizada verticalmente y centralizada horizontalmente.
    - (x) Descentralizada vertical y horizontalmente.
    - ( ) Descentralizada horizontalmente y centralizada verticalmente.
    - ( ) Centralizada vertical y horizontalmente.

    > Correcta: descentralizada vertical y horizontalmente (descentralización total tipo E). Que el poder llegue a los operarios del núcleo implica que baja por la jerarquía (vertical) y sale de la línea hacia los expertos (horizontal): es la burocracia profesional.

4. Indique cómo podría un directivo de primera línea incrementar el tamaño de la unidad que supervisa:

    - (x) Formalizando el comportamiento de sus subordinados.
    - ( ) Permitiendo que sus subordinados se coordinen mediante adaptación mutua.
    - ( ) Coordinando a sus subordinados a través de supervisión directa.
    - ( ) Un directivo de primera línea nunca podrá supervisar una unidad de gran tamaño.

    > Correcta: formalizando el comportamiento. La normalización (formalización) reduce la necesidad de supervisión directa y permite ampliar el tramo de control; la adaptación mutua y la supervisión directa lo reducen.

5. La hipótesis de configuración extendida pone de manifiesto que la estructura eficaz requiere:

    - ( ) Únicamente que los parámetros de diseño se ajusten debidamente a los factores de contingencia.
    - ( ) Únicamente que exista una consistencia interna entre los distintos parámetros de diseño.
    - (x) Una consistencia interna entre los distintos parámetros de diseño y la adecuación de estos parámetros de diseño a los factores de contingencia.
    - ( ) Según esta hipótesis no existe ninguna estructura eficaz.

    > Correcta: consistencia interna + adecuación a la contingencia. La hipótesis de configuración (ampliada) combina la congruencia (ajuste a factores de contingencia) con la coherencia interna entre parámetros de diseño.

6. ¿Qué características tienen las estructuras organizativas planas?:

    - ( ) Tramos de control estrechos y pocos niveles jerárquicos.
    - ( ) Tramos de control estrechos y muchos niveles jerárquicos.
    - (x) Tramos de control amplios y pocos niveles jerárquicos.
    - ( ) Tramos de control amplios y muchos niveles jerárquicos.

    > Correcta: tramos amplios y pocos niveles. Un mayor tramo de control reduce el número de niveles jerárquicos, aplanando la estructura.

7. La burocracia maquinal presenta:

    - ( ) Un sistema técnico poco regulador y automatizado.
    - ( ) Puestos de trabajo del núcleo de operaciones ampliados en ambas dimensiones.
    - ( ) Un núcleo de operaciones con autonomía para determinar la forma de realizar su trabajo.
    - (x) Una alta formalización en los puestos del núcleo de operaciones.

    > Correcta: alta formalización del núcleo. Su mecanismo de coordinación es la normalización de procesos: puestos muy especializados y formalizados, sin autonomía operativa.

8. Cuando el trabajo de los operarios se ve controlado y/o condicionado por el sistema técnico, se dice que el sistema técnico es:

    - ( ) Regulado.
    - ( ) Automatizado.
    - (x) Regulador.
    - ( ) Sofisticado.

    > Correcta: regulador. El carácter regulador mide en qué grado el sistema técnico controla o condiciona el trabajo de los operarios.

9. La estructura simple presenta:

    - ( ) Una amplia línea media y una tecnoestructura y staff de apoyo desarrollados.
    - ( ) Puestos de trabajo especializados y formalizados.
    - ( ) Una formulación de estrategias racional y analítica.
    - (x) Una división poco estricta del trabajo.

    > Correcta: división poco estricta del trabajo. La estructura simple es orgánica: poca especialización y formalización, línea media y tecnoestructura mínimas, y coordinación por supervisión directa del ápice.

10. ¿Cuál de las siguientes características se corresponden con las de una burocracia profesional?:

    - ( ) Su sistema técnico regulador.
    - ( ) Los puestos del núcleo de operaciones están muy formalizados, ya que es necesaria una elevada formación para ocuparlos.
    - ( ) La centralización de la estructura.
    - (x) Los puestos del núcleo de operaciones se encuentran especializados horizontalmente por conocimientos y ampliados verticalmente.

    > Correcta: especializados horizontalmente por conocimientos y ampliados verticalmente. El profesional domina una materia (especialización horizontal por conocimientos) y goza de gran autonomía (ampliación vertical); la coordinación es por normalización de habilidades, no por formalización.

11. Una de las ventajas de la burocracia maquinal es:

    - (x) Ser una estructura eficiente cuando se necesita realizar tareas sencillas y repetitivas.
    - ( ) Su adaptación rápida a los cambios del entorno.
    - ( ) El fuerte sentido de la misión de sus miembros.
    - ( ) La no existencia de problemas humanos en el núcleo de operaciones.

    > Correcta: eficiente para tareas sencillas y repetitivas. Es su gran ventaja; como contrapartida es rígida ante el cambio y genera problemas humanos por la fuerte formalización.

12. Analice la siguiente afirmación: "Debido a la naturaleza de su principal mecanismo de coordinación, la organización misionaria puede considerarse como una adhocracia ideológica":

    - ( ) Es falsa, se puede considerar como una burocracia profesional.
    - ( ) Es falsa, se puede considerar como una adhocracia administrativa.
    - (x) Es falsa, se puede considerar como una burocracia ideológica.
    - ( ) Es verdadera.

    > Correcta: es falsa, es una burocracia ideológica. La misionaria coordina por normalización de valores (estandarización), lo que la hace burocrática, no adhocrática (la adhocracia se basa en la adaptación mutua).

13. ¿En qué dispositivo de enlace se incumple el principio de unidad de mando?:

    - ( ) Puestos de enlace.
    - ( ) Grupos de trabajo.
    - ( ) Directivos integradores.
    - (x) Estructura matricial.

    > Correcta: estructura matricial. Es un sistema de doble mando (dos jefes con igual autoridad formal), rompiendo la unidad de mando.

14. ¿Qué dos parámetros de diseño facilitan el control externo de una organización?

    - ( ) La agrupación basada en unidades de mercado y los puestos profesionales en el núcleo de operaciones.
    - ( ) La existencia de dispositivos de enlace para la toma de decisiones y la descentralización vertical.
    - (x) La centralización y la formalización del comportamiento.
    - ( ) Ninguno, el diseño de la estructura organizativa no se ve influido por la existencia de un control externo.

    > Correcta: centralización y formalización. El control externo concentra el poder en el ápice (centralización) y exige normas claras (formalización) para hacer responsable a la dirección.

15. Analice la siguiente afirmación: "El núcleo de operaciones de una burocracia maquinal se coordina mediante normalización de procesos, provocando que la tecnoestructura sea la parte más importante de la organización":

    - ( ) Es falsa, el núcleo de operaciones de una burocracia maquinal se coordina mediante normalización de habilidades.
    - ( ) Es falsa, el staff de apoyo es la parte más importante de una burocracia maquinal.
    - ( ) Es falsa, el núcleo de operaciones de una burocracia maquinal se coordina mediante supervisión directa, provocando que sea el ápice estratégico la parte más importante de la organización.
    - (x) Es verdadera.

    > Correcta: es verdadera. La burocracia maquinal se coordina por normalización de procesos, diseñados por la tecnoestructura, que se convierte así en la parte clave (con poder informal).

16. Con respecto a la normalización como mecanismo de coordinación, señale la respuesta correcta:

    - ( ) La normalización de resultados no deja ninguna libertad de acción al trabajador para determinar la forma de realizar el trabajo.
    - ( ) La normalización que deja una mayor autonomía al trabajador es la de procesos de trabajo.
    - ( ) En la normalización de valores coincide la persona que ejecuta y administra la tarea.
    - (x) La formación necesaria para la normalización de habilidades puede ser adquirida por el trabajador antes de entrar a formar parte de la organización.

    > Correcta: la formación puede adquirirse antes de entrar (p. ej. en la universidad). La normalización de resultados fija el qué y deja libre el cómo; la de procesos es la que menos autonomía deja, no la que más.

17. La falta de cooperación interdisciplinaria entre los profesionales, con el consecuente problema para el desarrollo eficaz de innovaciones, es un problema típico de:

    - (x) La burocracia profesional.
    - ( ) La forma divisional.
    - ( ) La estructura simple.
    - ( ) La adhocracia.

    > Correcta: la burocracia profesional. Cada profesional trabaja con gran autonomía y poca cooperación interdisciplinar, lo que dificulta innovar (la innovación es precisamente la fortaleza de la adhocracia).

18. Indique cuál de las siguientes afirmaciones sobre las etapas de transición hacia una forma divisional pura es correcta:

    - ( ) En la forma puramente funcional la organización comercializa sus productos intermedios en el mercado.
    - ( ) En la forma de subproductos cada división atiende a sus propios mercados, ofreciendo líneas de productos no relacionados con las demás divisiones.
    - (x) En la forma de los productos relacionados las divisiones ofrecen productos diferentes pero relacionados entre sí, manteniendo algún aspecto en común.
    - ( ) En la forma conglomerada cada división se encarga de una función específica dentro del proceso de transformación que tiene lugar dentro de la organización.

    > Correcta: en la forma de productos relacionados las divisiones ofrecen productos distintos pero con un hilo común. En la forma funcional pura no se venden productos intermedios; el conglomerado agrupa por mercados independientes, no por funciones.

19. ¿Qué características tendrá el entorno de una estructura simple?:

    - (x) Sencillo y dinámico.
    - ( ) Sencillo y estable.
    - ( ) Complejo y estable.
    - ( ) Complejo y dinámico.

    > Correcta: sencillo y dinámico. El dinamismo impide normalizar (estructura orgánica) y la sencillez permite que una sola persona (ápice) controle por supervisión directa.

20. El personal directivo de una organización:

    - (x) Se distribuye entre la línea media, el ápice estratégico, la tecnoestructura y el staff de apoyo.
    - ( ) Se encuentra sólo en el ápice estratégico y la línea media.
    - ( ) Se distribuye entre la línea media y el núcleo de operaciones.
    - ( ) Ocupa exclusivamente la línea media.

    > Correcta: se distribuye entre línea media, ápice, tecnoestructura y staff de apoyo. Estas unidades de staff también tienen sus propios directivos; el núcleo de operaciones es la única parte sin personal directivo.

21. Analice la siguiente afirmación: "Como la estructura de las divisiones suele ser de burocracia maquinal, a la forma divisional también se la conoce con la denominación de burocracia duplicada":

    - ( ) Es falsa, la burocracia duplicada es una configuración híbrida entre la burocracia maquinal y la profesional.
    - ( ) Es falsa, las divisiones de la forma divisional nunca adoptan la forma de burocracia maquinal.
    - (x) Es falsa, la burocracia duplicada es una configuración híbrida entre la burocracia maquinal y la forma divisional.
    - ( ) Es verdadera.

    > Correcta: es falsa, la burocracia duplicada es un híbrido entre burocracia maquinal y forma divisional, no un sinónimo de la forma divisional.

22. Señale la afirmación correcta relacionada con el organigrama de una empresa:

    - ( ) El organigrama refleja todas las posibles relaciones que tienen lugar entre los miembros de la organización.
    - ( ) El organigrama representa la visión de la organización como un sistema de constelaciones de trabajo.
    - ( ) El organigrama permite identificar las unidades que se coordinan mediante adaptación mutua.
    - (x) El organigrama refleja gráficamente las relaciones de autoridad formal de la organización.

    > Correcta: refleja las relaciones de autoridad formal. El organigrama no muestra las relaciones informales (adaptación mutua) ni las constelaciones de trabajo.

23. Indique cuál de las siguientes afirmaciones relacionadas con la planificación de acciones es FALSA:

    - ( ) Los sistemas de planificación son diseñados por la tecnoestructura.
    - ( ) A través de la planificación de acciones se determinan las acciones necesarias para alcanzar los objetivos.
    - (x) La planificación de acciones es necesaria para coordinar unidades de mercado que actúan de manera independiente.
    - ( ) La planificación de acciones se considera un parámetro de diseño.

    > Falsa: las unidades de mercado independientes se coordinan por control del rendimiento (normalización de resultados), no por planificación de acciones, propia de las estructuras funcionales interdependientes.

24. Teniendo en cuenta la relación entre el entorno y la estructura organizativa, indique cuál de las siguientes afirmaciones es cierta:

    - (x) La sencillez y la extrema hostilidad del entorno conducen a una centralización de la estructura.
    - ( ) La complejidad y estabilidad del entorno conducen a una centralización de la estructura.
    - ( ) La disparidad y munificencia del entorno conducen a una centralización de la estructura.
    - ( ) La sencillez y la diversificación del entorno conducen a una centralización de la estructura.

    > Correcta: sencillez + extrema hostilidad → centralización. La hostilidad extrema obliga a centralizar para una respuesta rápida, y la sencillez permite que el ápice controle. La complejidad, en cambio, exige descentralizar.

25. En las estructuras divisionales puras ¿cómo se encuentra el poder?

    - ( ) Centralizado verticalmente en la sede central.
    - ( ) Descentralizado horizontalmente hacia los responsables de las distintas divisiones.
    - ( ) Descentralizado verticalmente, de forma selectiva, hacia los responsables de las distintas divisiones.
    - (x) Descentralizado verticalmente, de forma paralela, hacia los responsables de las distintas divisiones.

    > Correcta: descentralización vertical paralela (limitada). La sede delega un bloque amplio de decisiones de forma simultánea a todos los directores de división.

26. ¿Qué característica comparten una adhocracia administrativa y una adhocracia operativa?:

    - ( ) El núcleo de operaciones está truncado.
    - ( ) Se trabaja por encargo de los clientes.
    - (x) Su principal mecanismo de coordinación es la adaptación mutua.
    - ( ) Ambas organizaciones acabarán transformándose en estructuras simples.

    > Correcta: la adaptación mutua. Es el mecanismo común a ambas; el truncamiento del núcleo y el trabajo por encargo distinguen a la administrativa de la operativa, respectivamente.

27. La identificación natural del individuo con la ideología de una organización se produce cuando:

    - (x) Sus valores personales son congruentes con la ideología de la organización.
    - ( ) Se ve sometido a procesos de socialización por parte de la organización.
    - ( ) Se ve sometido a procesos de adoctrinamiento por parte de la organización.
    - ( ) El individuo únicamente se identifica con la ideología para obtener un beneficio personal.

    > Correcta: cuando sus valores coinciden con la ideología. La identificación natural surge de la congruencia de valores; la socialización y el adoctrinamiento son formas de identificación provocada (seleccionada/inculcada).

28. ¿Qué característica del entorno comparten la burocracia profesional y la adhocracia?:

    - ( ) Sencillo.
    - (x) Complejo.
    - ( ) Dinámico.
    - ( ) Diversificado.

    > Correcta: complejo. Ambas operan en entornos complejos; se diferencian en que la burocracia profesional vive en entorno estable y la adhocracia en entorno dinámico.

29. Analice la siguiente afirmación: "La agrupación de mercado puede favorecer el desarrollo de economías de escala, al concentrar en una única unidad los recursos (conocimientos) necesarios para llevar a cabo una determinada actividad que afecta a diferentes unidades":

    - ( ) Es verdadera.
    - (x) Es falsa, es la agrupación funcional la que puede favorecer el desarrollo de economías de escala.
    - ( ) Es falsa, es la agrupación profesional la que puede favorecer el desarrollo de economías de escala.
    - ( ) Es falsa, ninguna base de agrupación favorece el desarrollo de economías de escala.

    > Correcta: es falsa, son las economías de escala una ventaja de la agrupación funcional, que concentra recursos especializados. La agrupación de mercado, al contrario, los duplica entre unidades.

30. Analice la siguiente afirmación: "La socialización y el adoctrinamiento son dos procesos de naturaleza formal mediante los cuales se transmiten las normas de comportamiento relacionadas con la cultura, valores e ideología de la organización".

    - ( ) Es falsa, a través de estos procesos se transmiten los conocimientos y habilidades necesarios para desempeñar un puesto.
    - (x) Es falsa, la socialización es un proceso de naturaleza informal.
    - ( ) Es falsa, el adoctrinamiento es un proceso de naturaleza informal.
    - ( ) Es verdadera.

    > Correcta: es falsa, la socialización es informal, mientras que el adoctrinamiento es el proceso formal mediante el que la organización inculca sus valores.

31. Indique cuál de las siguientes afirmaciones relacionadas con el concepto de comité permanente es FALSA:

    - ( ) Es un dispositivo de enlace.
    - ( ) Es un ejemplo de constelación de trabajo.
    - ( ) Los individuos que forman parte de un comité permanente se relacionan entre sí a través de una intensa comunicación informal.
    - (x) Se trata de un conjunto de trabajadores que son convocados para cumplir una tarea determinada y específica, desagrupándose tras su cumplimiento.

    > Falsa: esa descripción corresponde al grupo de trabajo (temporal), no al comité permanente, que es estable en el tiempo.

32. La adhocracia emprendedora:

    - ( ) Es una configuración híbrida que se mueve entre la adhocracia operativa y la burocracia profesional.
    - (x) Es una configuración híbrida que se mueve entre la adhocracia operativa y la estructura simple.
    - ( ) Es una configuración híbrida que se mueve entre la adhocracia administrativa y la burocracia profesional.
    - ( ) Es una configuración híbrida que se mueve entre la adhocracia administrativa y la burocracia maquinal simple.

    > Correcta: híbrido entre adhocracia operativa y estructura simple. Combina la innovación por adaptación mutua con un fuerte liderazgo del ápice característico de la estructura simple.

33. ¿Qué característica tienen en común las estructuras simples y las adhocracias?:

    - ( ) Las características de su entorno.
    - ( ) El ápice estratégico concentra todo el poder.
    - (x) Son estructuras orgánicas.
    - ( ) Las estructuras simples y las adhocracias no tienen ninguna característica común.

    > Correcta: ambas son orgánicas (baja formalización). Difieren en el entorno (la simple, sencillo; la adhocracia, complejo) y en el reparto de poder.

34. Las organizaciones misionarias conversoras:

    - ( ) Lleva a cabo acciones para fomentar cambios radicales en la sociedad.
    - (x) Intentan cambiar la sociedad cambiando la conducta de sus propios miembros de acuerdo con su misión.
    - ( ) Prefieren aislarse de cualquier influencia externa y no pretenden cambiar la sociedad.
    - ( ) Son aquellas que se coordinan mediante normalización de procesos.

    > Correcta: cambian la sociedad cambiando a sus propios miembros. Las reformadoras buscan cambios radicales directos y las claustrales se aíslan del exterior.

35. Si el poder se encuentra repartido de forma igualitaria entre todos los miembros de la organización:

    - ( ) Existe descentralización vertical y centralización horizontal.
    - ( ) Existe descentralización horizontal y vertical hacia distintas constelaciones de trabajo.
    - ( ) Existe descentralización horizontal y vertical hacia el núcleo de operaciones.
    - (x) Existe descentralización total.

    > Correcta: descentralización total. El reparto igualitario del poder entre todos los miembros es la definición de descentralización total (organización democrática).

36. Marque la opción correcta:

    - ( ) Las unidades de línea se encargan de prestar soporte y ayuda para que las actividades primarias de la cadena de valor se realicen con éxito.
    - ( ) Los analistas del staff de apoyo se encargan de normalizar los procesos, habilidades y outputs de la organización.
    - ( ) La tecnoestructura, el staff de apoyo y la línea media se consideran unidades de línea.
    - (x) El staff de apoyo se encarga de realizar parte de las actividades secundarias de la cadena de valor, al margen del flujo fundamental del trabajo operativo.

    > Correcta: el staff de apoyo realiza actividades de apoyo (secundarias), fuera del flujo operativo. La normalización la hacen los analistas de la tecnoestructura, y la tecnoestructura y el staff son unidades de staff, no de línea.

37. Señale en qué configuración estructural se da una jerarquía denominada de "pirámide invertida":

    - ( ) Estructura simple.
    - ( ) Adhocracia.
    - (x) Burocracia profesional.
    - ( ) Esto no se da en ninguna configuración estructural.

    > Correcta: la burocracia profesional. Se describe como "pirámide invertida" por el gran poder de los operarios (profesionales), a los que los directivos sirven.

38. ¿Cómo se denomina el parámetro de diseño que introduce descripciones que indican en todo momento lo que han de hacer los trabajadores en sus puestos de trabajo?

    - ( ) Especialización.
    - ( ) Preparación.
    - (x) Formalización.
    - ( ) Adoctrinamiento.

    > Correcta: formalización del comportamiento. Normaliza los procesos de trabajo mediante reglas y descripciones que indican qué hacer en cada momento.

39. ¿Sobre qué parte de la organización recae el poder cuando existe centralización vertical y horizontal?

    - (x) El poder formal e informal recae sobre el ápice estratégico.
    - ( ) El poder formal e informal recae sobre los profesionales del núcleo de operaciones.
    - ( ) El poder formal recae sobre el ápice estratégico y el poder informal sobre la tecnoestructura.
    - ( ) El poder formal e informal recae sobre cualquier miembro de la organización.

    > Correcta: el ápice estratégico. La centralización vertical y horizontal concentra todo el poder (formal e informal) en la cúspide; es el caso de la estructura simple.

40. Con qué mecanismo de coordinación se relaciona la visión de la organización como un sistema de comunicación informal:

    - ( ) La supervisión directa.
    - (x) La adaptación mutua.
    - ( ) La normalización de procesos.
    - ( ) La normalización de valores.

    > Correcta: la adaptación mutua. La comunicación informal es la base del ajuste mutuo entre los miembros.

41. Un puesto implica una sola tarea pero sobre la que el operario tiene una gran libertad de acción, ¿cómo se encuentra?

    - ( ) Especializado horizontal y verticalmente.
    - (x) Especializado horizontalmente y ampliado verticalmente.
    - ( ) Ampliado horizontalmente y especializado verticalmente.
    - ( ) Ampliado horizontal y verticalmente.

    > Correcta: especializado horizontalmente (una sola tarea) y ampliado verticalmente (gran libertad/control sobre ella). Es el perfil del puesto profesional.

42. ¿Cómo son los puestos profesionales?

    - ( ) Especializados horizontalmente (tareas) y ampliado verticalmente.
    - (x) Especializados horizontalmente (conocimientos) y ampliado verticalmente.
    - ( ) Ampliado horizontalmente (tareas) y ampliado verticalmente.
    - ( ) Especializados horizontalmente (conocimientos) y verticalmente.

    > Correcta: especializados horizontalmente por conocimientos y ampliados verticalmente. El profesional se especializa en un área de conocimiento, pero tiene gran autonomía (ampliación vertical).

43. Un directivo medio tiene autoridad sobre un responsable de primera línea y sólo responde ante el gerente.

    - ( ) El puesto del directivo está más ampliado verticalmente que los otros dos.
    - ( ) El puesto del gerente está más especializado verticalmente que los otros.
    - ( ) El puesto del directivo tiene más especialización vertical que los otros dos.
    - (x) El puesto del responsable de línea es el más especializado verticalmente.

97. La forma divisional pura se caracteriza por:

    - ( ) Un cierto grado de descentralización vertical selectiva desde la sede central hacia las divisiones y una descentralización del poder de toma de decisiones en las propias divisiones.

    > Correcta: descentralización vertical paralela (limitada) de la sede hacia las divisiones, que internamente tienden a centralizar el poder en su director.

98. El núcleo de operaciones de la burocracia operativa:

    - (x) Se encuentra fusionado con el resto de la tecnoestructura formando una "masa orgánica" en la que no está clara la diferencia entre la línea y staff.

    - ( ) Está automatizado y separado de la estructura.

    > Correcta: fusionado en una masa orgánica donde no se distingue línea de staff. En la adhocracia operativa el trabajo directivo y operativo se combinan; el núcleo truncado y separado es propio de la adhocracia administrativa.

99. Teniendo en cuenta las hipótesis de congruencia, indique cuál de las siguientes afirmaciones es cierta.

    - (x) La complejidad y la virtualización están en consonancia con una estructura red.
    - ( ) La simpleza y la virtualización están en consonancia con una estructura simple.

    > Correcta: complejidad + virtualización ↔ estructura red. La estructura simple es orgánica (sin formalización ni estandarización), por lo que las demás combinaciones son incongruentes.

100. En las estructuras divisionales puras, ¿cuál es el principal mecanismo de coordinación entre las divisiones?
