# EXAMEN OE 21-22 - Organización de Empresas

* **Descripción:** Banco de preguntas tipo test del examen 21-22 de Organización de Empresas (basado en la teoría de Mintzberg), con las respuestas correctas marcadas y razonadas.
* **Titulación:** Grado en Administración y Dirección de Empresas / Ingeniería Informática


1. Con respecto a la organización como un sistema de flujos regulados, señale la afirmación correcta:

    - (x) La visión de la organización como un sistema de flujos regulados considera, entre otros, a los flujos horizontales que tienen lugar entre los especialistas del staff y tecnoestructura con los directivos de línea media.
    - ( ) Los flujos regulados incluyen exclusivamente flujos verticales descendentes, que representan las órdenes que fluyen desde el ápice estratégico hasta el núcleo de operaciones.
    - ( ) La visión de la organización como un sistema de flujos regulados es un reflejo completo de todas las relaciones de flujo que tienen lugar dentro de la organización.
    - ( ) La visión de la organización como un sistema de flujos regulados solo considera el flujo horizontal dentro del núcleo de operaciones.

    > El sistema de flujos regulados describe el uso formal de la autoridad e información. Esto incluye el flujo de trabajo operativo, el flujo de control regulado y los flujos de información y asesoramiento (staff y tecnoestructura hacia la línea media).

2. Un puesto de trabajo que está especializado horizontalmente:

    - ( ) Siempre estará muy formalizado.
    - ( ) Está obligatoriamente ampliado verticalmente.
    - ( ) No puede exigir formación para ser realizado.
    - (x) Realiza pocas tareas o tareas con poco contenido.

    > La especialización horizontal se refiere a la amplitud del puesto; un puesto muy especializado horizontalmente engloba una gama muy reducida de tareas.

3. Considerando una organización en la que todo el poder recae en los profesionales del núcleo de operaciones, indique la opción correcta:

    - ( ) La organización se encuentra descentralizada verticalmente y centralizada horizontalmente.
    - (x) La organización se encuentra descentralizada vertical y horizontalmente.
    - ( ) La organización se encuentra descentralizada horizontalmente y centralizada verticalmente.
    - ( ) La organización se encuentra centralizada vertical y horizontalmente.

    > Es el caso típico de la Burocracia Profesional. El poder desciende hasta el núcleo de operaciones (descentralización vertical) y recae sobre los propios profesionales operativos, no sobre administradores ni analistas (descentralización horizontal).

4. Indique cómo un directivo de primera línea podría incrementar el tamaño de la unidad que supervisa:

    - (x) Formalizando el comportamiento de sus subordinados.
    - ( ) Permitiendo que sus subordinados se coordinen mediante adaptación mutua.
    - ( ) Coordinando a sus subordinados a través de supervisión directa.
    - ( ) Un directivo de primera línea nunca podrá supervisar una unidad de gran tamaño.

    > La estandarización (o normalización/formalización) del trabajo permite que un solo supervisor controle a muchos más subordinados, ampliando el tramo de control.

5. La hipótesis de configuración extendida pone de manifiesto que la estructura eficaz requiere:

    - ( ) Únicamente que los parámetros de diseño se ajusten debidamente a los factores de contingencia.
    - ( ) Únicamente que exista una consistencia interna entre los distintos parámetros de diseño.
    - (x) Una consistencia interna entre los distintos parámetros de diseño y la adecuación de estos parámetros de diseño a los factores de contingencia.
    - ( ) Según esta hipótesis no existe ninguna estructura eficaz.

    > Mintzberg postula que la eficacia organizacional exige tanto coherencia interna (entre los parámetros de diseño) como externa (con los factores de contingencia).

6. ¿Qué características tienen las estructuras organizativas planas?

    - ( ) Tramos de control estrechos y pocos niveles jerárquicos.
    - ( ) Tramos de control estrechos y muchos niveles jerárquicos.
    - (x) Tramos de control amplios y pocos niveles jerárquicos.
    - ( ) Tramos de control amplios y muchos niveles jerárquicos.

    > Una estructura plana surge cuando los tramos de control son amplios, lo que reduce la necesidad de crear múltiples capas directivas jerárquicas intermedias.

7. La burocracia maquinal presenta:

    - (x) Un sistema técnico regulador y poco sofisticado.
    - ( ) Puestos de trabajo del núcleo de operaciones ampliados en ambas dimensiones.
    - ( ) Una formulación de estrategias intuitiva.
    - ( ) Una división poco estricta del trabajo.

    > En la burocracia maquinal el sistema técnico marca el ritmo y rige el trabajo rutinario (es regulador), pero no suele estar totalmente automatizado (poco sofisticado).

8. Cuando el trabajo de los operarios se ve controlado y/o condicionado por el sistema técnico, se dice que el sistema técnico es...

    - ( ) Sofisticado.
    - ( ) Automatizado.
    - (x) Regulador.
    - ( ) Complejo.

    > Un sistema técnico "regulador" es aquel que impone el ritmo y estandariza el trabajo del operario, limitando su autonomía.

9. La estructura simple presenta:

    - ( ) Un entorno sencillo y estable.
    - ( ) Puestos de trabajo especializados y formalizados.
    - ( ) Una formulación de estrategias racional y analítica.
    - (x) Una división poco estricta del trabajo.

    > La estructura simple carece de tecnoestructura y de formalización, lo que se traduce en una división del trabajo flexible y poco estricta donde todos hacen un poco de todo bajo la supervisión directa del líder. Su entorno es sencillo y dinámico.

10. Uno de los inconvenientes de la burocracia profesional se encuentra en:

    - ( ) Su sistema técnico regulador.
    - ( ) La formalización del comportamiento de los profesionales.
    - ( ) La centralización de la estructura.
    - (x) Su dificultad para la innovación.

    > La burocracia profesional está diseñada para perfeccionar programas estándar y encasillar los problemas conocidos (ej. hospitales, universidades), por lo que le resulta muy difícil crear respuestas nuevas ante problemas inéditos (innovación).

11. Una de las ventajas de la burocracia maquinal es:

    - (x) Ser una estructura eficiente cuando se necesita realizar tareas sencillas y repetitivas.
    - ( ) Su adaptación rápida a los cambios del entorno.
    - ( ) El fuerte sentido de la misión de sus miembros.
    - ( ) La no existencia de problemas humanos en el núcleo de operaciones.

    > Su diseño está concebido como una máquina para producir de forma masiva y con alta eficiencia en entornos estables.

12. Analice la siguiente afirmación: "Debido a la naturaleza de su principal mecanismo de coordinación, la organización misionaria puede considerarse como una adhocracia ideológica".

    - ( ) Es falsa, se puede considerar como una burocracia profesional.
    - ( ) Es falsa, se puede considerar como una adhocracia administrativa.
    - (x) Es falsa, se puede considerar como una burocracia ideológica.
    - ( ) Es verdadera.

    > Falsa. La organización misionaria coordina mediante la normalización de normas y valores. Esta fuerte estandarización de las creencias la asemeja más a una forma de "burocracia ideológica" y no a la adhocracia, que es fluida y orgánica.

13. ¿Qué dispositivo de enlace consiste en un conjunto de trabajadores que son convocados para cumplir una determinada tarea y se desagrupa tras su cumplimiento?

    - ( ) Puestos de enlace.
    - (x) Grupos de trabajo.
    - ( ) Directivos integradores.
    - ( ) Estructura matricial permanente.

    > También conocidos como "Task forces". Son temporales y se disuelven cuando resuelven el problema específico.

14. Teniendo en cuenta las hipótesis de congruencia, cuanto mayor sea el control externo de una organización...

    - ( ) más descentralizada y formalizada resulta su estructura.
    - ( ) más descentralizada y menos formalizada resulta su estructura.
    - (x) más centralizada y formalizada resulta su estructura.
    - ( ) más descentralizada y menos formalizada resulta su estructura.

    > Cuando la organización está muy controlada desde el exterior (ej. filiales, entes públicos), el ápice estratégico retiene el poder para rendir cuentas (centralización) y estandariza al máximo para asegurar el cumplimiento (formalización).

15. La burocracia maquinal se caracteriza por un diseño del sistema decisor:

    - ( ) Centralizado vertical y horizontalmente.
    - ( ) Centralizado verticalmente y descentralizado horizontalmente hacia el staff de apoyo.
    - (x) Centralizado verticalmente y descentralizado horizontalmente hacia la tecnoestructura.
    - ( ) Centralizado verticalmente y descentralizado horizontalmente hacia el núcleo de operaciones.

    > El poder formal está centralizado en el ápice, pero los analistas de la tecnoestructura ganan mucho poder informal (descentralización horizontal) porque son ellos quienes diseñan las reglas que todos obedecen.

16. Con respecto a la normalización como mecanismo de coordinación, señale la respuesta correcta:

    - ( ) La normalización de resultados no deja libertad de acción al trabajador para determinar la forma de realizar el trabajo.
    - ( ) La normalización que deja una mayor autonomía al trabajador es la de procesos de trabajo y la que menos la de habilidades.
    - ( ) En la normalización de procesos coincide la persona que ejecuta y administra la tarea.
    - (x) Mediante la normalización de habilidades, la organización sacrifica parte del control en beneficio de las entidades externas que forman a sus trabajadores.

    > Al requerir habilidades estandarizadas (ej. título de medicina), la organización cede el control del proceso de formación a universidades y colegios profesionales.

17. En las burocracias profesionales ¿suelen surgir dos jerarquías paralelas?:

    - ( ) No, en las burocracias profesionales no pueden existir dos jerarquías paralelas.
    - ( ) Sí, una democrática y ascendente para los profesionales del staff de apoyo y otra con forma de burocracia maquinal para la tecnoestructura.
    - (x) Sí, una democrática y ascendente para los profesionales de las unidades de línea y otra con forma de burocracia maquinal para el staff de apoyo.
    - ( ) Sí, una forma de burocracia maquinal en el núcleo de operaciones y otra democrática y ascendente en los profesionales del staff y la tecnoestructura.

    > Los profesionales del núcleo (ej. médicos) operan de forma democrática y autónoma, mientras que el personal de apoyo administrativo (staff) está sujeto a un control rígido, jerárquico y centralizado.

18. La forma divisional pura se caracteriza por:

    - ( ) Una centralización total en la sede central.
    - ( ) Un cierto grado de descentralización horizontal paralela desde la sede central hacia las divisiones.
    - (x) Un cierto grado de descentralización vertical paralela desde la sede central hacia las divisiones y una centralización del poder de toma de decisiones en las propias divisiones.
    - ( ) Un cierto grado de descentralización vertical selectiva desde la sede central hacia las divisiones y una descentralización del poder de toma de decisiones en las propias divisiones.

    > La sede cede a los gerentes divisionales la autoridad (descentralización vertical paralela), pero estos suelen retener el poder en sus propias divisiones sin descentralizarlo más hacia abajo.

19. El núcleo de operaciones de la adhocracia operativa:

    - (x) Se encuentra fusionado con el resto de la estructura formando una "masa orgánica" en la que no está clara la diferencia entre línea y staff.
    - ( ) Se encuentra tronchado y separado de la estructura.
    - ( ) Se coordina mediante la normalización de procesos.
    - ( ) Está automatizado y separado de la estructura.

    > En la adhocracia operativa, los operarios trabajan en proyectos junto a los directivos y expertos del staff, disolviendo las fronteras tradicionales de la organización.

20. El personal directivo de una organización:

    - ( ) Se distribuye entre la línea media, el ápice estratégico, la tecnoestructura y el staff de apoyo.
    - (x) Se encuentra solo en el ápice estratégico y la línea media.
    - ( ) Se distribuye entre la línea media y el núcleo de operaciones.
    - ( ) Ocupa exclusivamente la línea media.

    > El personal directivo constituye la cadena de autoridad formal (cadena de mando), que en el modelo de Mintzberg engloba exclusivamente el ápice estratégico y la línea media.

21. Analice la siguiente afirmación: "Como la estructura de cada una de las divisiones suele ser de burocracia maquinal, a la forma divisional también se la conoce con la denominación de burocracia duplicada".

    - ( ) Es falsa, la burocracia duplicada es una configuración híbrida entre la burocracia maquinal y la profesional.
    - ( ) Es falsa, las divisiones de la forma divisional nunca adoptan la forma de burocracia duplicada.
    - ( ) Es falsa, la burocracia duplicada es una configuración híbrida entre la burocracia maquinal y la forma divisional.
    - (x) Es verdadera.

    > Mintzberg define la configuración divisional muchas veces como "burocracias de copia al carbón" (burocracias duplicadas), ya que cada división suele adoptar la estructura de una burocracia maquinal independiente.

22. Señale la afirmación correcta relacionada con el organigrama de una empresa:

    - (x) El organigrama refleja gráficamente las relaciones de autoridad formal de la organización y proporciona una adecuada representación de la división del trabajo que hay en ella.

    > El organigrama es estrictamente la representación del esqueleto formal, mostrando quién rinde cuentas a quién (autoridad) y cómo se agrupan los puestos (división del trabajo).

23. Indique cuál de las siguientes afirmaciones relacionadas con la planificación de acciones es FALSA:

    - (x) La planificación de acciones permite introducir en la estructura el mecanismo de coordinación de la normalización de resultados de forma directa.

24. Teniendo en cuenta las hipótesis de congruencia, indique cuál de las siguientes afirmaciones es cierta:

    - (x) La sencillez y extrema hostilidad del entorno conducen a una centralización de la estructura.

    - ( ) La sencillez y munificencia del entorno conducen a una centralización de la estructura.

    > Ante un entorno extremadamente hostil (crisis severa), la organización necesita una respuesta rápida y coordinada, por lo que el poder se centraliza fuertemente en la cúspide (ápice estratégico).

25. En las estructuras divisionales puras ¿cuál es el principal mecanismo para coordinar a las divisiones?

    - (x) Normalización de resultados, para conservar el control y proteger la autonomía divisional.

    > La sede central concede autonomía operativa a las divisiones, pero evalúa y controla su rendimiento a través de la estandarización de los resultados esperados (ej. cuotas de rentabilidad).

26. Los principales mecanismos de coordinación de la adhocracia operativa y de la adhocracia administrativa:

    > La adhocracia es la configuración más orgánica; confía plenamente en la comunicación informal y el ajuste recíproco (adaptación mutua) entre expertos para realizar proyectos innovadores.

27. La identificación natural del individuo con la ideología de una organización se produce cuando:

28. ¿Qué tipo de entorno comparten la estructura simple y la adhocracia?

    - ( ) Sencillo.
    - ( ) Complejo.

29. ¿Qué interdependencias se consiguen controlar usando una base de agrupación según mercado?

    > Al agrupar por mercado (clientes, productos o áreas), se encierran dentro de la misma unidad todas las fases secuenciales del proceso para crear ese producto o atender a ese cliente (flujo de trabajo).

30. Analice la siguiente afirmación: "La socialización y el adoctrinamiento son dos procesos de naturaleza formal mediante los cuales se transmiten las normas de comportamiento relacionadas con la cultura, valores e ideología de la organización".

    > El adoctrinamiento es el parámetro de diseño formal de la empresa, mientras que la socialización hace referencia al proceso natural e informal por el que un individuo asimila la cultura grupal.

31. Indique cuál de las siguientes afirmaciones relacionadas con el concepto de estructura matricial es FALSA:

    - ( ) Es un dispositivo de enlace.
    - ( ) Existen dos tipos de estructuras matriciales, permanentes y variables.

    - (x) Hay dos tipos de estructuras matriciales, unas que se agrupan sólo en base a la función y otras que se agrupan sólo en base al mercado.

    > Falsa. La característica fundamental de la estructura matricial es que rompe el principio de unidad de mando creando un "doble mando", combinando de forma *simultánea* la agrupación funcional y la agrupación por mercado.

32. La adhocracia emprendedora:

    - ( ) Es una configuración híbrida que se mueve entre la adhocracia operativa y la burocracia profesional.

    - ( ) Es una configuración híbrida que se mueve entre la adhocracia administrativa y la burocracia maquinal simple.

    > Combina la fluidez y el talento experto (adhocracia) con el liderazgo fuerte y centralizado de un visionario (estructura simple), muy típica en startups tecnológicas.

33. A mayor tamaño de la organización:

    - ( ) Menor será la necesidad de división del trabajo, menos especializadas estarán sus tareas, menos diferenciadas sus unidades y más desarrollado su componente administrativo.
    - ( ) Mayor será la necesidad de división del trabajo, más especializadas estarán sus tareas, menos diferenciadas sus unidades y más desarrollado su componente administrativo.
    - (x) Mayor será la necesidad de división del trabajo, más especializadas estarán sus tareas, más diferenciadas sus unidades y más desarrollado su componente administrativo.
    - ( ) Menor será la necesidad de división del trabajo, menos especializadas estarán sus tareas, menos diferenciadas sus unidades y menos desarrollado su componente administrativo.

    > Conforme la organización crece, se vuelve más compleja: divide más las tareas, especializa, crea nuevos departamentos (diferenciación) y requiere más staff/jerarquía para coordinarlos (componente administrativo).

34. Las organizaciones misionarias reformadoras:

    - ( ) Se orientan a fomentar cambios radicales en la sociedad.
    - (x) Intentan cambiar la sociedad cambiando la conducta de sus propios miembros de acuerdo con su misión.
    - ( ) Prefieren aislarse de cualquier influencia externa.
    - ( ) Son una variante de la estructura simple.

    > A diferencia de las conversoras (cambio exterior y proselitismo activo) o las de claustro (aislamiento), las reformadoras proponen el cambio social progresivo mediante el ejemplo y la rectitud de conducta de sus miembros.

35. Si el poder se encuentra repartido de forma igualitaria entre todos los miembros de la organización:

    - ( ) Existe descentralización vertical y centralización horizontal.
    - ( ) Existe descentralización horizontal y centralización vertical.
    - ( ) Existe descentralización horizontal y vertical.
    - (x) Existe descentralización total.

    > La descentralización es total cuando el poder fluye desde la cúspide a todos los niveles inferiores (vertical) y además recae en los operarios y no en staff o analistas (horizontal), asemejándose a un entorno asambleario o muy orgánico.

36. Marque la opción correcta:

    - ( ) Las unidades de línea se encargan de prestar soporte y ayuda para que las actividades primarias de la cadena de valor se realicen con éxito.
    - ( ) Los analistas del staff de apoyo se encargan de normalizar los procesos, habilidades y outputs de la organización.

    - (x) El staff de apoyo se encarga de realizar parte de las actividades secundarias de la cadena de valor, al margen del flujo fundamental del trabajo operativo.

    > Servicios como cafetería, limpieza, relaciones públicas o asesoría legal operan al margen del flujo central de la organización y se integran en el staff de apoyo.

37. Analice la siguiente afirmación: "En las burocracias profesionales, los miembros de la línea media suelen ser elegidos, o al menos consentidos, por los operarios del núcleo de operaciones".

    - ( ) Falso, esto ocurre en la adhocracia administrativa.
    - ( ) Falso, esto ocurre en la burocracia maquinal.

    > Dado el enorme poder de los profesionales (ej. catedráticos en una universidad o médicos jefes en un hospital), los administradores de línea media suelen requerir el beneplácito de los operarios para poder coordinarlos.

38. ¿Cómo se denomina el parámetro de diseño que introduce descripciones que indican en todo momento lo que han de hacer los trabajadores en sus puestos de trabajo?

    - ( ) Especialización.
    - ( ) Preparación.
    - (x) Formalización.
    - ( ) Adoctrinamiento.

    > La formalización del comportamiento persigue estandarizar los procesos de trabajo mediante reglas, manuales de procedimiento y descripciones exhaustivas de puestos.

39. ¿Sobre qué parte de la organización recae el poder cuando existe centralización vertical y horizontal?

    - (x) El poder formal e informal recae sobre el ápice estratégico.
    - ( ) El poder formal e informal recae sobre los profesionales del núcleo de operaciones.
    - ( ) El poder formal recae sobre el ápice estratégico y el poder informal sobre la tecnoestructura.
    - ( ) El poder formal e informal recae sobre cualquier miembro de la organización.

    > En la centralización estricta (típica de la Estructura Simple), el líder o gerente superior retiene todo el poder en la toma de decisiones, sin delegarlo hacia abajo ni a los lados.

40. La visión de la organización como un sistema de comunicación informal describe el uso de:

    - ( ) La supervisión directa.
    - (x) La adaptación mutua.
    - ( ) La normalización de procesos.
    - ( ) La normalización de valores.

    > El funcionamiento informal escapa a las reglas documentadas y recae sobre redes de contactos e intercambio espontáneo de información (sociometría) para coordinar el trabajo (adaptación mutua).
