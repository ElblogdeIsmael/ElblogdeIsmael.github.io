    - (x) Verdadero
    - ( ) Falso

<!-- TEMA 2: Análisis y diseño del puesto de trabajo -->

1. La especificación o perfil del puesto recoge la información relativa a cada puesto en relación con las competencias del trabajador (conocimientos, habilidades, experiencia y formación necesarias).

    - (x) Falso

    > Falso. Eso es la especificación del puesto. La descripción recoge tareas, obligaciones, responsabilidades y condiciones laborales.

    - ( ) Verdadero

    - ( ) Verdadero
    - (x) Falso

    > Falso. No tienen por qué tener una jornada laboral reducida.

6. La reingeniería es la reconsideración fundamental y el rediseño radical de los procesos de la empresa para lograr mejoras trascendentales en desempeño, coste, calidad, servicio y rapidez; se hace en torno a resultados y afecta a la descripción de puestos.

    - (x) Verdadero
    - ( ) Falso

    - ( ) Verdadero
    - (x) Falso

    > Falso. Recoger la opinión de especialistas corresponde a la técnica de grupos de expertos, no a la observación.

9. El análisis de puestos de trabajo es el proceso a través del cual la empresa recopila y analiza la información sobre el puesto para identificar las tareas, obligaciones y responsabilidades del puesto.

11. Cuando el análisis de puestos implica redactar la descripción teniendo en cuenta no solo tareas y responsabilidades, sino también conocimientos, experiencias y competencias, se refiere al análisis multifuncional.

    - (x) Verdadero
    - ( ) Falso

12. Ese mismo análisis (que incluye conocimientos, experiencias y competencias) se refiere a la reingeniería.

    - ( ) Verdadero
    - (x) Falso

    > Falso. La reingeniería es rediseño radical de procesos. Lo descrito es el análisis multifuncional.

13. Entre las técnicas utilizadas para realizar el rediseño de puestos se encuentran la simplificación del puesto, la ampliación del puesto y la reestructuración del puesto.

    - ( ) Verdadero

    > Falso. La reestructuración no es una técnica de rediseño de puestos.

14. Entre las técnicas utilizadas para realizar el rediseño de puestos se encuentran la simplificación del puesto, la ampliación del puesto y el enriquecimiento del puesto.

    - (x) Verdadero

    - ( ) Verdadero
    - (x) Falso

    > Falso. Se solucionan a través de la ampliación o el enriquecimiento del puesto, no de la simplificación.

    - ( ) Verdadero
    - (x) Falso

<!-- TEMA 3 y 10: Planificación de los RRHH y procesos sustractivos -->

    - ( ) Verdadero
    - (x) Falso

    - (x) Falso

    - (x) Verdadero
    - ( ) Falso

6. La demanda de RRHH de una empresa no se puede planificar.

    - ( ) Verdadero
    - (x) Falso

    - ( ) Verdadero

    > Falso. La empresa que despide se hace cargo íntegramente del coste; no se descuenta de la indemnización.

10. El programa de outplacement o recolocación es un programa creado para ayudar a los empleados despedidos a superar el estrés emocional y a buscar nuevo empleo.

    - (x) Verdadero
    - ( ) Falso

    - ( ) Verdadero

12. Los planes de RRHH se pueden clasificar en planes a largo, medio y corto plazo.

    - (x) Verdadero
    - ( ) Falso

    - (x) Verdadero

14. La ruptura laboral voluntaria está formada por la dimisión y los despidos.

    - ( ) Verdadero
    - (x) Falso

    > Falso. La ruptura voluntaria es dimisión, jubilación y jubilación anticipada. Los despidos son ruptura involuntaria.

15. La ruptura laboral involuntaria incluye despidos, reducciones de plantilla y jubilación anticipada.

    - ( ) Verdadero
    - (x) Falso

    > Falso. La involuntaria son despidos (individuales o colectivos) y reducciones de plantilla; la jubilación anticipada es voluntaria.

16. La ruptura laboral voluntaria comprende la dimisión, la jubilación y la jubilación anticipada.

    - (x) Verdadero
    - ( ) Falso

17. La ruptura laboral supone la desvinculación del empleado con la empresa y puede ser voluntaria o involuntaria.

    - (x) Verdadero

    - ( ) Verdadero

19. La excedencia es una supresión laboral.

    - ( ) Verdadero
    - (x) Falso

    > Falso. La excedencia es una suspensión laboral (temporal). La supresión es la eliminación del puesto de trabajo.

20. Entre las alternativas a la reducción de plantillas se proponen contratar nuevos empleados, renovar contratos que finalizan y aumentar las horas extras.

    - ( ) Verdadero
    - (x) Falso

    > Falso. Esas medidas son contraproducentes para reducir plantilla. Una alternativa válida es incentivar excedencias voluntarias.

21. Para obtener información sobre la oferta externa de trabajo se usan las auditorías de RRHH en las empresas.

    - ( ) Verdadero
    - (x) Falso

    > Falso. Las auditorías de RRHH dan información sobre la oferta INTERNA (resumen de habilidades y conocimientos de los empleados).

22. Los organigramas recogen el porcentaje de empleados que mantienen su puesto año a año.

    - ( ) Verdadero
    - (x) Falso

    > Falso. Eso es el análisis de Markov. Los organigramas representan la estructura organizativa (ocupantes y necesidades futuras de personal).

23. Un ERE es un proceso sustractivo de reducción de plantilla y, por tanto, una ruptura laboral involuntaria.

    - (x) Verdadero
    - ( ) Falso

24. La excedencia debida a la designación o elección para un cargo público o a funciones sindicales de ámbito provincial o superior se considera excedencia forzosa (obligatoria).

    - (x) Verdadero
    - ( ) Falso

    > Verdadero. Es una excedencia forzosa/obligatoria que la empresa siempre debe conceder al trabajador.

    - (x) Falso
