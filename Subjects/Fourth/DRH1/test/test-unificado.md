    - ( ) Verdadero

<!-- TEMA 3: La planificación de los RRHH -->

    - (x) Falso

    > Falso. La de RRHH asegura el número suficiente de personal, con la cualificación adecuada y en el momento oportuno; la empresarial cubre aspectos económicos, fuerzas de mercado, etc. La de RRHH complementa a la empresarial.

    - ( ) Falso

5. La administración del personal se preocupa por la planificación de plantilla.

    - (x) Verdadero
    - ( ) Falso

    - (x) Verdadero
    - ( ) Falso

7. La alta tasa de paro en un país hace que aumente la oferta de personal.

    - (x) Verdadero
    - ( ) Falso

8. Las jubilaciones son uno de los factores que modifican la demanda de personal.

    - (x) Verdadero
    - ( ) Falso

9. La demanda de RRHH de una empresa no se puede planificar.

    - ( ) Verdadero
    - (x) Falso

    - ( ) Verdadero
    - (x) Falso

    - ( ) Verdadero
    - (x) Falso

    > Falso. Esas medidas son contraproducentes para reducir plantilla. Una alternativa válida es incentivar excedencias voluntarias.

13. Incentivar a los empleados a que voluntariamente tomen excedencias es una alternativa a la reducción de plantillas.

    - (x) Verdadero
    - ( ) Falso

17. La ruptura laboral supone la desvinculación del empleado con la empresa y puede ser voluntaria o involuntaria.

    - (x) Verdadero
    - ( ) Falso

18. La ruptura laboral voluntaria comprende la dimisión, la jubilación y la jubilación anticipada.

19. La ruptura laboral involuntaria incluye despidos, reducciones de plantilla y jubilación anticipada.

    - ( ) Verdadero

    - ( ) Verdadero
    - (x) Falso

    - ( ) Verdadero
    - (x) Falso

22. La excedencia debida a la designación o elección para un cargo público o a funciones sindicales de ámbito provincial o superior se considera excedencia forzosa (obligatoria).

    - (x) Falso

    - (x) Verdadero
    - ( ) Falso

    > Verdadero. El ERTE suspende temporalmente el contrato (el vínculo no se rompe y hay derecho a reincorporarse); el ERE es ruptura laboral (despido colectivo o extinción).

26. El programa de outplacement o recolocación se crea para ayudar a los empleados despedidos a superar el estrés emocional y a buscar nuevo empleo, y su coste lo asume la empresa que despide.

    - (x) Verdadero
    - ( ) Falso

    > Verdadero. Minimizar las demandas judiciales es otro de sus beneficios. El coste no se descuenta de la indemnización.

27. Para obtener información sobre la oferta interna de trabajo se usan las auditorías de RRHH, que ofrecen un resumen de las habilidades y conocimientos de los empleados.

    - (x) Verdadero
    - ( ) Falso

    > Verdadero. Las auditorías de RRHH informan de la oferta INTERNA, no de la tasa de empleo del país.

28. Los organigramas recogen el porcentaje de empleados que mantienen su puesto año a año.

    - ( ) Verdadero
    - (x) Falso

    > Falso. Eso es el análisis de Markov. Los organigramas representan la estructura organizativa (ocupantes y necesidades futuras de personal).

29. La técnica Delphi aplicada a la planificación de RRHH es una técnica cualitativa presencial en la que se discute abiertamente la evolución de la oferta y la demanda de RRHH entre expertos.

    - ( ) Verdadero
    - (x) Falso

    > Falso. La técnica Delphi es cualitativa NO presencial y anónima (cuestionarios sucesivos) para evitar la presión de grupo. La que es presencial y con debate es la técnica de grupo nominal.

30. La presión de grupo al utilizar la técnica Delphi puede hacer que las valoraciones de los expertos sean menores de lo esperado.

    - ( ) Verdadero
    - (x) Falso

    > Falso. Ese efecto ocurre en la técnica de Grupo Nominal (debate cara a cara). La Delphi es anónima precisamente para evitarlo.

31. La previsión de la demanda de RRHH puede realizarse con métodos cuantitativos como el análisis de tendencias y métodos cualitativos como el diagrama de dispersión.

    - ( ) Verdadero
